## 🔑 第2章 必须掌握的内容

### 2.1 解析函数概念（必考判断/选择）

**关键定义链**：
- 函数在一点**可导** → 不一定解析
- 函数在一点**解析** → 在该点的某邻域内处处可导
- 函数在区域D内**解析** ↔ 在D内每一点可导

**常见误区（样卷反复考）**：
- ✗ "可导必解析" → 错，只有邻域内处处可导才行
- ✗ "满足C-R条件必解析" → 错，还需偏导数连续
- ✓ "解析必有任意阶导数"

### 2.2 柯西-黎曼方程（C-R方程）⭐核心

直角坐标：
\[
\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}, \quad 
\frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}
\]

极坐标：
\[
\frac{\partial u}{\partial r} = \frac{1}{r}\frac{\partial v}{\partial \theta}, \quad
\frac{\partial v}{\partial r} = -\frac{1}{r}\frac{\partial u}{\partial \theta}
\]

**充要条件**：\(f(z)=u+iv\) 在区域D内解析 ↔ \(u,v\) 在D内可微且满足C-R方程。

### 2.3 初等函数（必考填空）

| 函数 | 公式 | 周期性 | 注意事项 |
|------|------|--------|----------|
| \(e^z\) | \(e^z = e^x(\cos y + i\sin y)\) | \(2\pi i\) | \(|e^z|=e^x\) |
| \(\sin z\) | \(\sin z = \frac{e^{iz}-e^{-iz}}{2i}\) | \(2\pi\) | \(|\sin z|\) 可以 > 1 |
| \(\cos z\) | \(\cos z = \frac{e^{iz}+e^{-iz}}{2}\) | \(2\pi\) | 同上 |
| \(\text{Ln}\,z\) | \(\text{Ln}\,z = \ln|z| + i\arg z + 2k\pi i\) | — | 多值函数，主值 \(\ln z\) |
| \(z^\alpha\) | \(z^\alpha = e^{\alpha\text{Ln}\,z}\) | — | 一般多值 |
| 反三角函数 | 用对数表示 | — | 了解即可 |

**必记特殊值**：
- \(e^{i\pi}=-1\), \(e^{i\pi/2}=i\)
- \(\text{Ln}(-1) = (2k+1)\pi i\), 主值 \(\ln(-1)=\pi i\)
- \(\text{Ln}(1+i) = \ln\sqrt{2} + i\frac{\pi}{4} + 2k\pi i\)

### 2.4 调和函数与解析函数的关系

- 若 \(f=u+iv\) 解析，则 \(u,v\) 均为调和函数（满足拉普拉斯方程 \(u_{xx}+u_{yy}=0\)）
- 已知 \(u\)（调和），求共轭调和函数 \(v\)：利用C-R方程积分
- 已知 \(u\)，构造解析函数：\(f(z) = u + i\int (u_x dy - u_y dx)\) 或用 \(f'(z)=u_x-iu_y\) 积分

**必做习题**（从2.pdf）：
- 判断题：全部 —— 概念的精确辨析
- 第7题：用C-R条件证明解析性/非解析性
- 第8题：极坐标C-R条件
- 第9题：计算 \(e^{2+i}, i^i, (-2)^2\)
- 第10题：计算 \(\text{Ln}(1+i), 1^{\sqrt{2}}\)
- 第19题：若 \(f(z)\) 和 \(\overline{f(z)}\) 都在D内解析，则 \(f(z)\) 为常数

# 📘 Day 3-4：第3章 复变函数的积分

> ⚠️ **这是整个课程的工具核心**，柯西积分公式是后面所有章节的基础！

## 🔑 必须掌握的内容

### 3.1 基本定理与公式（必考填空/选择）

**柯西-古萨定理**：
若 \(f(z)\) 在单连通区域D内解析，则对D内任意闭曲线C：
\[
\oint_C f(z)\,dz = 0
\]

**柯西积分公式** ⭐最关键：
\[
f(z_0) = \frac{1}{2\pi i}\oint_C \frac{f(z)}{z-z_0}\,dz
\]

**高阶导数公式** ⭐必须背：
\[
f^{(n)}(z_0) = \frac{n!}{2\pi i}\oint_C \frac{f(z)}{(z-z_0)^{n+1}}\,dz
\]

**重要积分值（必背）**：
\[
\oint_{|z-a|=r} \frac{dz}{z-a} = 2\pi i
\]
\[
\oint_{|z-a|=r} \frac{dz}{(z-a)^n} = 0 \quad (n \geq 2, n \in \mathbb{Z})
\]

### 3.2 计算方法总结

| 方法 | 适用情形 |
|------|----------|
| 直接参数化 | 简单路径（直线、圆弧） |
| 柯西定理 | 被积函数解析，闭曲线 |
| 柯西积分公式 | 分母为 \(z-z_0\)（一次） |
| 高阶导数公式 | 分母为 \((z-z_0)^n\)（n≥2） |
| 复合闭路定理 | 多连通区域 |

### 3.3 典型题目套路

**样卷高频题**：计算单位圆周上的积分
- 例如：\(\displaystyle\oint_{|z|=1} \frac{\cos z + e^z}{z}dz\) → 用柯西积分公式 = \(2\pi i(\cos 0 + e^0)=4\pi i\)
- 例如：\(\displaystyle\oint_{|z|=1} \frac{z^3+3z+1}{(z-a)^3}dz\) → 用高阶导数公式

**必做习题**（从3.pdf）：
- 填空题：全部（4,5,7,8,9特别重要）
- 第1题：\(\int_{-i}^{i} |z|\,dz\) 不同路径
- 第6题：计算 \(\oint_{|z|=2} \frac{dz}{z^4-1}\)
- 第8-10题：多值函数分支积分
- 第11题：\(\int_0^1 \frac{dz}{1+z^2}\)
- 第16-19题：各种闭曲线积分
- 第21题：求导数值

# 📘 Day 5-6：第4章 解析函数的级数表示

## 🔑 必须掌握的内容

### 4.1 幂级数收敛半径

**求法**（必考填空/选择）：
- 比值法：\(R = \lim_{n\to\infty} \left|\frac{a_n}{a_{n+1}}\right|\)
- 根值法：\(R = \frac{1}{\limsup_{n\to\infty}\sqrt[n]{|a_n|}}\)

### 4.2 泰勒展开 ⭐

**常见函数的麦克劳林展开（必背）**：

| 函数 | 展开式 | 收敛半径 |
|------|--------|----------|
| \(e^z\) | \(\sum_{n=0}^{\infty} \frac{z^n}{n!}\) | \(\infty\) |
| \(\sin z\) | \(\sum_{n=0}^{\infty} (-1)^n \frac{z^{2n+1}}{(2n+1)!}\) | \(\infty\) |
| \(\cos z\) | \(\sum_{n=0}^{\infty} (-1)^n \frac{z^{2n}}{(2n)!}\) | \(\infty\) |
| \(\frac{1}{1-z}\) | \(\sum_{n=0}^{\infty} z^n\) | \(1\) |
| \(\text{Ln}(1+z)\) | \(\sum_{n=1}^{\infty} (-1)^{n-1}\frac{z^n}{n}\) | \(1\) |
| \((1+z)^\alpha\) | \(1+\alpha z + \frac{\alpha(\alpha-1)}{2!}z^2+\cdots\) | \(1\) |

### 4.3 罗朗展开 ⭐⭐（必考大题）

**方法**：将函数在指定圆环内展开为 \(\sum_{n=-\infty}^{\infty} c_n (z-z_0)^n\)

**核心技巧**：利用 \(\frac{1}{1-z} = \sum_{n=0}^{\infty} z^n\)（当 \(|z|<1\)）和 \(\frac{1}{1-z} = -\sum_{n=1}^{\infty} \frac{1}{z^n}\)（当 \(|z|>1\)）

**典型例题**：
- \(f(z)=\frac{1}{(z-1)(z-2)}\) 在 \(1<|z|<2\) 内展开
- \(f(z)=\frac{1}{(z+1)(z+2)}\) 在 \(1<|z|<2\) 内展开

### 4.4 孤立奇点分类 ⭐⭐

| 类型 | 罗朗展开特点 | \(\lim_{z\to z_0}f(z)\) |
|------|-------------|------------------------|
| 可去奇点 | 无负幂项 | 有限值 |
| m阶极点 | 最高负幂为 \(-m\) | \(\infty\) |
| 本性奇点 | 无穷多负幂项 | 不存在 |

**零点与极点的关系**：若 \(z_0\) 是 \(f(z)\) 的 m 阶零点，则 \(z_0\) 是 \(\frac{1}{f(z)}\) 的 m 阶极点。

**必做习题**（从4.pdf）：
- 判断题：全部
- 第3-8题：求收敛半径
- 第12-13题：泰勒展开
- 第18-21题：罗朗展开（重点！）
- 第22-24题：孤立奇点分类
- 第26题：零点与极点关系证明

# 📘 Day 7：第5章 留数理论（§5.1）⭐⭐拿分核心

> §5.2 不考！只复习§5.1内容。

## 🔑 必须掌握的内容

### 5.1 留数定义与计算

**定义**：\(f(z)\) 在 \(z_0\) 的留数 = 罗朗展开中 \(\frac{1}{z-z_0}\) 项的系数 \(c_{-1}\)

记为：\(\text{Res}[f(z), z_0]\)

**留数计算规则（必背）**：

| 情形 | 公式 |
|------|------|
| \(z_0\) 为可去奇点 | \(\text{Res}[f(z),z_0]=0\) |
| \(z_0\) 为1阶极点 | \(\text{Res}[f(z),z_0]=\lim_{z\to z_0}(z-z_0)f(z)\) |
| \(f(z)=\frac{P(z)}{Q(z)}\)，\(Q(z_0)=0, Q'(z_0)\neq 0\) | \(\text{Res}[f(z),z_0]=\frac{P(z_0)}{Q'(z_0)}\) |
| \(z_0\) 为m阶极点 | \(\text{Res}[f(z),z_0]=\frac{1}{(m-1)!}\lim_{z\to z_0}\frac{d^{m-1}}{dz^{m-1}}[(z-z_0)^m f(z)]\) |
| 无穷远点 | \(\text{Res}[f(z),\infty]=-\text{Res}\left[\frac{1}{z^2}f\left(\frac{1}{z}\right), 0\right]\) |

### 5.2 留数定理

\[
\oint_C f(z)\,dz = 2\pi i \sum_{k=1}^n \text{Res}[f(z), z_k]
\]
其中 \(z_k\) 为C内部的所有孤立奇点。

也可用无穷远点：
\[
\sum \text{Res}[f(z), \text{所有有限奇点}] + \text{Res}[f(z), \infty] = 0
\]

### 5.3 用留数计算实积分 ⭐⭐⭐必考大题

**类型一**：\(\int_0^{2\pi} R(\cos\theta, \sin\theta)\,d\theta\)
- 令 \(z=e^{i\theta}\)，则 \(\cos\theta = \frac{z+z^{-1}}{2}\), \(\sin\theta = \frac{z-z^{-1}}{2i}\), \(d\theta = \frac{dz}{iz}\)
- 化为 \(\oint_{|z|=1} f(z)\,dz = 2\pi i\sum\text{Res}\)

**类型二**：\(\int_{-\infty}^{\infty} \frac{P(x)}{Q(x)}\,dx\)（\(Q(x)\) 比 \(P(x)\) 高至少2次，分母无实根）
- = \(2\pi i\sum \text{Res}[f(z), \text{上半平面奇点}]\)

**类型三**：\(\int_{-\infty}^{\infty} \frac{P(x)}{Q(x)}e^{imx}\,dx\)（含三角函数的反常积分）
- 用约当引理

**必做习题**（从5.pdf）：
- 填空题：全部（1-10）
- 第1-5题：求留数
- 第6-8题：用留数求围道积分
- 第12-14题：实积分计算（类型一、二）
- 第15-16题：含三角函数的反常积分

# 📘 Day 8：第7章 傅里叶变换

## 🔑 必须掌握的内容

### 7.1 定义（必背）

**傅里叶变换**：
\[
F(\omega) = \mathcal{F}[f(t)] = \int_{-\infty}^{\infty} f(t) e^{-i\omega t}\,dt
\]

**傅里叶逆变换**：
\[
f(t) = \mathcal{F}^{-1}[F(\omega)] = \frac{1}{2\pi}\int_{-\infty}^{\infty} F(\omega) e^{i\omega t}\,d\omega
\]

### 7.2 单位脉冲函数 δ(t)

- \(\int_{-\infty}^{\infty} \delta(t)\,dt = 1\)
- \(\mathcal{F}[\delta(t)] = 1\)
- \(\mathcal{F}[1] = 2\pi\delta(\omega)\)
- 筛选性：\(\int_{-\infty}^{\infty} f(t)\delta(t-t_0)\,dt = f(t_0)\)

### 7.3 重要性质（必考填空/计算）

| 性质 | \(f(t)\) | \(F(\omega)\) |
|------|----------|---------------|
| 线性 | \(af+bg\) | \(aF+bG\) |
| 时移 | \(f(t-t_0)\) | \(F(\omega)e^{-i\omega t_0}\) |
| 频移 | \(f(t)e^{i\omega_0 t}\) | \(F(\omega-\omega_0)\) |
| 微分 | \(f'(t)\) | \(i\omega F(\omega)\) |
| 积分 | \(\int_{-\infty}^t f(\tau)d\tau\) | \(\frac{F(\omega)}{i\omega}+\pi F(0)\delta(\omega)\) |
| 对称 | \(F(t)\) | \(2\pi f(-\omega)\) |

### 7.4 卷积

\[
f(t)*g(t) = \int_{-\infty}^{\infty} f(\tau)g(t-\tau)\,d\tau
\]
\[
\mathcal{F}[f*g] = F(\omega)G(\omega)
\]

### 7.5 常见傅里叶变换对（必背）

| \(f(t)\) | \(F(\omega)\) |
|----------|---------------|
| \(e^{-\alpha|t|}\) (\(\alpha>0\)) | \(\frac{2\alpha}{\alpha^2+\omega^2}\) |
| 矩形脉冲（门函数） | 抽样函数形式 |
| \(\delta(t)\) | \(1\) |
| \(1\) | \(2\pi\delta(\omega)\) |

**必做题型**（依据样卷）：
- 计算具体函数的傅里叶变换（如样卷1第9题）
- 用傅里叶变换解微分方程（样卷2第10题）

# 📘 Day 9：第8章 拉普拉斯变换 ⭐⭐拿分核心

> 样卷固定最后一大题10分：用拉普拉斯变换解ODE！

## 🔑 必须掌握的内容

### 8.1 定义

\[
F(s) = \mathcal{L}[f(t)] = \int_0^{\infty} f(t) e^{-st}\,dt
\]

### 8.2 常见函数的拉普拉斯变换（必背！）

| \(f(t)\) | \(F(s)\) | 收敛域 |
|----------|----------|--------|
| \(1\) | \(\frac{1}{s}\) | \(\text{Re}\,s > 0\) |
| \(t^n\) | \(\frac{n!}{s^{n+1}}\) | \(\text{Re}\,s > 0\) |
| \(e^{at}\) | \(\frac{1}{s-a}\) | \(\text{Re}\,s > a\) |
| \(\sin\omega t\) | \(\frac{\omega}{s^2+\omega^2}\) | \(\text{Re}\,s > 0\) |
| \(\cos\omega t\) | \(\frac{s}{s^2+\omega^2}\) | \(\text{Re}\,s > 0\) |
| \(\delta(t)\) | \(1\) | — |
| \(u(t)\)（单位阶跃） | \(\frac{1}{s}\) | \(\text{Re}\,s > 0\) |

### 8.3 重要性质

| 性质 | \(f(t)\) | \(F(s)\) |
|------|----------|----------|
| 线性 | \(af+bg\) | \(aF+bG\) |
| 时移 | \(f(t-a)u(t-a)\) | \(e^{-as}F(s)\) |
| 频移 | \(e^{at}f(t)\) | \(F(s-a)\) |
| 微分（关键！） | \(f'(t)\) | \(sF(s)-f(0)\) |
| 二阶微分 | \(f''(t)\) | \(s^2F(s)-sf(0)-f'(0)\) |
| 积分 | \(\int_0^t f(\tau)d\tau\) | \(\frac{F(s)}{s}\) |
| 卷积 | \(f(t)*g(t)\) | \(F(s)G(s)\) |

### 8.4 拉普拉斯逆变换

**方法**：
1. 部分分式分解（最常用）
2. 查表法
3. 卷积定理

**典型套路**：
\[
F(s) = \frac{1}{(s-1)(s-2)} \to f(t) = e^{2t} - e^t
\]
\[
F(s) = \frac{1}{s^2(s+1)} \to \text{部分分式分解后查表}
\]

### 8.5 用拉普拉斯变换解ODE（必考大题！）

**步骤**：
1. 方程两边取拉普拉斯变换
2. 利用微分性质代入初始条件
3. 解出 \(Y(s)\)
4. 求逆变换得 \(y(t)\)

**样卷1第10题**：\(y'' - y' = 2e^t\), \(y(0)=y'(0)=0\)
**样卷2第10题**：\(y'' - y' + y = f(t)\) （用傅里叶变换）

# 📋 必背公式速记卡（考前反复看）

## 复变函数核心公式

```
1. C-R方程：u_x = v_y,  u_y = -v_x
2. 柯西积分公式：∮ f(z)/(z-z₀) dz = 2πi·f(z₀)
3. 高阶导数：∮ f(z)/(z-z₀)^{n+1} dz = 2πi·f^{(n)}(z₀)/n!
```

## <a id="21"></a>§2.1 复变函数的概念、极限与连续性

### 一、复变函数的定义

**定义 2.1.1（复变函数）** 设 \(G \subset \mathbb{C}\) 是一个非空点集。若存在一个对应法则 \(f\)，使得对 \(G\) 中每一个复数 \(z = x + iy\)，都有唯一确定的复数 \(w = u + iv\) 与之对应，则称 \(f\) 是定义在 \(G\) 上的一个**复变函数**，记作

\[
w = f(z), \quad z \in G
\]

其中 \(z\) 称为**自变量**，\(w\) 称为**因变量**，\(G\) 称为函数的**定义域**。

**核心理解**：复变函数本质上是二元实向量函数。因为：
- 输入 \(z = x + iy\) 由两个实数 \((x, y)\) 决定
- 输出 \(w = u + iv\) 由两个实数 \((u, v)\) 决定

因此，一个复变函数 \(f(z)\) 等价于两个**二元实变函数**：

\[
f(z) = f(x+iy) = u(x,y) + i \cdot v(x,y)
\]

其中：
- \(u(x, y) = \operatorname{Re}[f(z)]\) —— 实部
- \(v(x, y) = \operatorname{Im}[f(z)]\) —— 虚部

> **例如**：\(f(z) = z^2 = (x+iy)^2 = (x^2-y^2) + i(2xy)\)，则 \(u(x,y)=x^2-y^2\)，\(v(x,y)=2xy\)。

**定义 2.1.2（单值函数与多值函数）** 若对每个 \(z \in G\)，\(f(z)\) 只有一个值，则称 \(f\) 为**单值函数**；若有多个值，则为**多值函数**。如 \(\operatorname{Ln} z\) 是多值函数，但可通过限制辐角范围得到**单值解析分支**。

---

### 二、复变函数的极限

**定义 2.1.3（极限）** 设函数 \(f(z)\) 在 \(z_0\) 的某去心邻域内有定义，\(A\) 为复常数。若对任意 \(\varepsilon > 0\)，存在 \(\delta > 0\)，使得当 \(0 < |z - z_0| < \delta\) 时，有

\[
|f(z) - A| < \varepsilon
\]

则称当 \(z\) 趋于 \(z_0\) 时 \(f(z)\) 以 \(A\) 为极限，记作

\[
\lim_{z \to z_0} f(z) = A
\]

**定理 2.1.1（极限的实部虚部分解）** 设 \(f(z) = u(x,y) + iv(x,y)\)，\(A = a + ib\)，\(z_0 = x_0 + iy_0\)，则

\[
\lim_{z \to z_0} f(z) = A \quad \Longleftrightarrow \quad \lim_{(x,y) \to (x_0,y_0)} u(x,y) = a \quad \text{且} \quad \lim_{(x,y) \to (x_0,y_0)} v(x,y) = b
\]

这是复变函数极限的**化归原则**：复极限等价于两个实二元极限同时存在且等于对应值。

**极限的四则运算法则**：若 \(\lim_{z\to z_0} f(z) = A\)，\(\lim_{z\to z_0} g(z) = B\)，则

1. \(\lim [f(z) \pm g(z)] = A \pm B\)
2. \(\lim [f(z) \cdot g(z)] = A \cdot B\)
3. \(\lim [f(z)/g(z)] = A/B\)（当 \(B \neq 0\)）

**重要性质**：
- 若 \(\lim_{z\to z_0} f(z)\) 存在，则极限唯一。
- \(\lim_{z\to z_0} |f(z)| = |\lim_{z\to z_0} f(z)|\)。

---

### 三、复变函数的连续性

**定义 2.1.4（连续性）** 设 \(f(z)\) 在 \(z_0\) 的某邻域内有定义。若

\[
\lim_{z \to z_0} f(z) = f(z_0)
\]

则称 \(f(z)\) 在 \(z_0\) 处**连续**。若 \(f(z)\) 在区域 \(D\) 内每一点都连续，则称 \(f(z)\) 在 \(D\) 内连续。

**定理 2.1.2（连续性的实部分解）** \(f(z) = u+iv\) 在 \(z_0 = x_0+iy_0\) 连续，当且仅当 \(u(x,y)\) 和 \(v(x,y)\) 均在 \((x_0, y_0)\) 连续。

**连续函数的性质**：
- 连续函数的和、差、积、商（分母不为零）仍连续。
- 连续函数的复合仍连续。
- 若 \(f(z)\) 在有界闭集上连续，则 \(f(z)\) 在该闭集上有界，且能达到最大模和最小模（**最值定理**）。
- 若 \(f(z)\) 在有界闭集上连续，则 \(f(z)\) 在该闭集上**一致连续**。

## <a id="22"></a>§2.2 解析函数的概念

### 一、复变函数的导数

**定义 2.2.1（导数）** 设函数 \(f(z)\) 在 \(z_0\) 的某邻域内有定义。若极限

\[
\lim_{\Delta z \to 0} \frac{f(z_0 + \Delta z) - f(z_0)}{\Delta z}
\]

存在（且为有限复常数），则称 \(f(z)\) 在 \(z_0\) 处**可导**（或**可微**），该极限值称为 \(f(z)\) 在 \(z_0\) 处的**导数**，记作

\[
f'(z_0) = \frac{df}{dz}\bigg|_{z=z_0}
\]

**关键区别**：实变函数中，\(\Delta x \to 0\) 只有两种方式（从左或从右）；而复变函数中，\(\Delta z \to 0\) 有**无穷多种方式**（沿任意曲线、任意方向）。因此复可导的条件比实可导**苛刻得多**。

**求导法则**（与实变函数形式相同）：
1. \((cf)' = cf'\)
2. \((f \pm g)' = f' \pm g'\)
3. \((fg)' = f'g + fg'\)
4. \((f/g)' = (f'g - fg')/g^2\)（\(g \neq 0\)）
5. 链式法则：\(\frac{d}{dz}[f(g(z))] = f'(g(z)) \cdot g'(z)\)

---

### 二、解析函数的定义

**定义 2.2.2（解析函数 / 全纯函数）**
- 若 \(f(z)\) 在 \(z_0\) 的**某邻域内处处可导**，则称 \(f(z)\) 在 \(z_0\) 处**解析**（analytic / holomorphic）。
- 若 \(f(z)\) 在区域 \(D\) 内每一点都解析，则称 \(f(z)\) 在 \(D\) 内**解析**。
- 若 \(f(z)\) 在整个复平面 \(\mathbb{C}\) 上解析，则称 \(f(z)\) 为**整函数**（entire function）。

**关键区分：**

| 概念 | 条件 | 例子 |
|------|------|------|
| 在一点**可导** | 仅在该点的导数定义式极限存在 | \(f(z) = |z|^2\) 仅在 \(z=0\) 可导 |
| 在一点**解析** | 在该点的**某个邻域内**处处可导 | 需要邻域条件 |
| 在区域**解析** | 在区域内每一点可导 | \(e^z, \sin z, z^n\) |

> 🔴 **高频考点**：一点可导 ≠ 一点解析！解析要求邻域条件。\(f(z) = |z|^2\) 在 \(z=0\) 可导但**不解析**，因为它在 \(z=0\) 的任何邻域内不是处处可导。

**定理 2.2.1（解析函数的性质）**
1. 解析函数的和、差、积、商（分母不为零）仍解析。
2. 解析函数的复合仍解析。
3. 若 \(f'(z) = 0\) 在区域 \(D\) 内恒成立，则 \(f(z)\) 在 \(D\) 内为常数。
4. 解析函数**具有任意阶导数**（这是复变函数独有的深刻性质）。

## <a id="23"></a>§2.3 函数可导与解析的充要条件

> 这是整个第二章的**理论核心**，也是考试的重中之重。

### 一、柯西-黎曼方程（C-R 方程）

**定理 2.3.1（可导的必要条件——C-R 方程）** 设 \(f(z) = u(x,y) + iv(x,y)\) 在 \(z_0 = x_0 + iy_0\) 处可导，则在 \((x_0, y_0)\) 处，\(u, v\) 的偏导数存在，且满足：

\[
\boxed{\frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}}, \quad
\boxed{\frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}}
\]

同时，导数可表示为：

\[
f'(z_0) = \frac{\partial u}{\partial x} + i\frac{\partial v}{\partial x}
       = \frac{\partial v}{\partial y} - i\frac{\partial u}{\partial y}
       = \frac{\partial u}{\partial x} - i\frac{\partial u}{\partial y}
       = \frac{\partial v}{\partial y} + i\frac{\partial v}{\partial x}
\]

**推导过程（理解记忆）**：

因为 \(\Delta z = \Delta x + i\Delta y\)，导数定义为：
\[
f'(z_0) = \lim_{\Delta z \to 0} \frac{\Delta u + i\Delta v}{\Delta x + i\Delta y}
\]

- **沿实轴接近**（\(\Delta y = 0\)，即 \(\Delta z = \Delta x\)）：
  \[
  f'(z_0) = \lim_{\Delta x \to 0} \frac{u(x_0+\Delta x, y_0)-u(x_0,y_0)}{\Delta x} + i\lim_{\Delta x \to 0} \frac{v(x_0+\Delta x, y_0)-v(x_0,y_0)}{\Delta x}
          = u_x(x_0, y_0) + i \cdot v_x(x_0, y_0)
  \]

- **沿虚轴接近**（\(\Delta x = 0\)，即 \(\Delta z = i\Delta y\)）：
  \[
  f'(z_0) = \lim_{\Delta y \to 0} \frac{u(x_0, y_0+\Delta y)-u(x_0,y_0)}{i\Delta y} + i\lim_{\Delta y \to 0} \frac{v(x_0, y_0+\Delta y)-v(x_0,y_0)}{i\Delta y}
          = \frac{1}{i}u_y + v_y = -i u_y + v_y = v_y - i u_y
  \]

两种方式得到的导数必须相等：
\[
u_x + i v_x = v_y - i u_y
\]

实部与虚部分别相等，即得 C-R 方程。

---

**定理 2.3.2（可导的充分条件）** 设 \(f(z) = u(x,y) + iv(x,y)\) 在 \(z_0 = x_0 + iy_0\) 的某邻域内有定义。若：
1. \(u, v\) 在 \((x_0, y_0)\) 处**可微**（即具有连续偏导数），
2. \(u, v\) 在 \((x_0, y_0)\) 处满足 C-R 方程，

则 \(f(z)\) 在 \(z_0\) 处**可导**。

---

**定理 2.3.3（解析的充要条件）** 设 \(f(z) = u(x,y) + iv(x,y)\) 在区域 \(D\) 内有定义。则 \(f(z)\) 在 \(D\) 内解析的**充要条件**是：

1. \(u(x,y), v(x,y)\) 在 \(D\) 内处处可微（通常只需验证偏导数连续），
2. 在 \(D\) 内处处满足 C-R 方程 \(\displaystyle \frac{\partial u}{\partial x} = \frac{\partial v}{\partial y}, \quad \frac{\partial u}{\partial y} = -\frac{\partial v}{\partial x}\)。

> 🔴 **常见错误**：仅验证 C-R 方程成立就断言解析，这是不充分的，必须同时验证 \(u, v\) 的可微性（通常验证一阶偏导数连续即可）。

---

### 二、C-R 方程的极坐标形式

当用极坐标表示时：\(z = re^{i\theta}\)，\(f(z) = u(r,\theta) + iv(r,\theta)\)

**极坐标 C-R 方程**：

\[
\boxed{\frac{\partial u}{\partial r} = \frac{1}{r} \frac{\partial v}{\partial \theta}}, \quad
\boxed{\frac{\partial v}{\partial r} = -\frac{1}{r} \frac{\partial u}{\partial \theta}}
\]

导数公式：
\[
f'(z) = e^{-i\theta}\left(\frac{\partial u}{\partial r} + i\frac{\partial v}{\partial r}\right)
      = \frac{1}{r} e^{-i\theta}\left(\frac{\partial v}{\partial \theta} - i\frac{\partial u}{\partial \theta}\right)
\]

**推导思路**：利用 \(x = r\cos\theta\)，\(y = r\sin\theta\)，通过链式法则将直角坐标的 C-R 方程转化为极坐标形式。

---

### 三、调和函数与共轭调和函数

**定义 2.3.1（调和函数）** 若二元实函数 \(\varphi(x,y)\) 在区域 \(D\) 内具有连续的二阶偏导数，且满足**拉普拉斯方程**：

\[
\boxed{\Delta \varphi = \frac{\partial^2 \varphi}{\partial x^2} + \frac{\partial^2 \varphi}{\partial y^2} = 0}
\]

则称 \(\varphi(x,y)\) 为 \(D\) 内的**调和函数**。

**定理 2.3.4（解析函数的实部与虚部是调和函数）** 若 \(f(z) = u+iv\) 在区域 \(D\) 内解析，则 \(u(x,y)\) 和 \(v(x,y)\) 都是 \(D\) 内的调和函数。

**证明思路**：由 C-R 方程：\(u_x = v_y\)，\(u_y = -v_x\)。
两边分别再求偏导：
- \(\frac{\partial}{\partial x}(u_x) = \frac{\partial}{\partial x}(v_y) \Rightarrow u_{xx} = v_{yx}\)
- \(\frac{\partial}{\partial y}(u_y) = \frac{\partial}{\partial y}(-v_x) \Rightarrow u_{yy} = -v_{xy}\)

由于 \(v_{yx} = v_{xy}\)（偏导数连续性保证），相加得 \(u_{xx} + u_{yy} = 0\)。同理可证 \(v_{xx} + v_{yy} = 0\)。

**定义 2.3.2（共轭调和函数）** 若 \(u\) 和 \(v\) 都是区域 \(D\) 内的调和函数，且满足 C-R 方程，则称 \(v\) 是 \(u\) 的**共轭调和函数**。

注意：\(u\) 和 \(v\) 的地位不对称——\(v\) 是 \(u\) 的共轭调和函数，但 \(-u\) 才是 \(v\) 的共轭调和函数。

---

### 四、已知实部求解析函数（考试重点题型）

**问题**：已知调和函数 \(u(x,y)\)，求以 \(u\) 为实部的解析函数 \(f(z) = u + iv\)。

**解法一（偏积分法）**：
1. 由 C-R 方程：\(v_x = -u_y\)，\(v_y = u_x\)。
2. 对 \(v_x = -u_y\) 关于 \(x\) 积分得 \(v = \int (-u_y) \,dx + \varphi(y)\)。
3. 对上述结果关于 \(y\) 求偏导，利用 \(v_y = u_x\) 确定 \(\varphi'(y)\)，再积分得 \(\varphi(y)\)。
4. 最终 \(f(z) = u + iv\)。

**解法二（利用导数公式）**：
\[
f'(z) = u_x + iv_x = u_x - iu_y
\]
将右边的 \(x, y\) 用 \(z\) 和 \(\bar{z}\) 表达（利用 \(x = \frac{z+\bar{z}}{2}, y = \frac{z-\bar{z}}{2i}\)），若 \(f'(z)\) 仅依赖于 \(z\) 而不含 \(\bar{z}\)，则可直接对 \(z\) 积分得到 \(f(z)\)。

## <a id="24"></a>§2.4 初等函数

> 初等函数是指数函数、三角函数、对数函数、幂函数等在复数域中的推广。

### 一、指数函数 \(e^z\)

**定义 2.4.1** 对任意 \(z = x + iy\)，定义

\[
\boxed{e^z = e^{x+iy} = e^x (\cos y + i\sin y)}
\]

**基本性质**：
1. **模**：\(|e^z| = e^x = e^{\operatorname{Re}z}\)。注意：\(|e^z|\) 仅取决于实部！
2. **辐角**：\(\arg(e^z) = y + 2k\pi = \operatorname{Im}z + 2k\pi\)。
3. **周期性**：\(e^{z+2\pi i} = e^z\)，周期为 \(T = 2\pi i\)。这是复数域独有的——实数指数函数无周期性！
4. **加法公式**：\(e^{z_1+z_2} = e^{z_1} \cdot e^{z_2}\)。
5. **处处解析**：\((e^z)' = e^z\)。
6. **\(e^z \neq 0\)** 对任何 \(z\) 成立（指数函数永不为零）。

**欧拉公式的特例**（必记）：
\[
e^{i\pi} = -1, \quad e^{i\pi/2} = i, \quad e^{2\pi i} = 1
\]

### 二、三角函数

**定义 2.4.2** 对任意复数 \(z\)，

\[
\boxed{\sin z = \frac{e^{iz} - e^{-iz}}{2i}}, \quad
\boxed{\cos z = \frac{e^{iz} + e^{-iz}}{2}}
\]

**基本性质**：
1. **解析性**：\(\sin z\) 和 \(\cos z\) 在整个复平面上解析（整函数）。
2. **导数**：\((\sin z)' = \cos z\)，\((\cos z)' = -\sin z\)。
3. **周期性**：\(\sin(z+2\pi) = \sin z\)，\(\cos(z+2\pi) = \cos z\)。
4. **恒等式**：\(\sin^2 z + \cos^2 z = 1\)；和差公式、倍角公式在复数域中仍成立。
5. **零点**：
   - \(\sin z = 0 \Longleftrightarrow z = k\pi\)
   - \(\cos z = 0 \Longleftrightarrow z = \frac{\pi}{2} + k\pi\)

> 🔴 **极易出错！** 在复数域中，\(\sin z\) 和 \(\cos z\) **是无界的**！
> 
> 例如：\(\cos(i) = \frac{e^{-1} + e^{1}}{2} \approx 1.543 > 1\)。
> 
> 样卷选择题经常在此设陷阱：“下列结论错误的是”——选项“\(|\cos z| \leq 1\)”就是错的。

**其他三角函数**：
\[
\tan z = \frac{\sin z}{\cos z}, \quad \cot z = \frac{\cos z}{\sin z}, \quad \sec z = \frac{1}{\cos z}, \quad \csc z = \frac{1}{\sin z}
\]

它们在其分母不为零处解析，求导法则与实变函数一致。

### 三、双曲函数

**定义 2.4.3**
\[
\boxed{\sinh z = \frac{e^z - e^{-z}}{2}}, \quad
\boxed{\cosh z = \frac{e^z + e^{-z}}{2}}
\]

**与三角函数的关系**（必记）：
\[
\sinh(iz) = i\sin z, \quad \cosh(iz) = \cos z
\]
\[
\sin(iz) = i\sinh z, \quad \cos(iz) = \cosh z
\]

**双曲函数的模无界性与三角函数类似**。

### 四、对数函数 \(\operatorname{Ln} z\)

**定义 2.4.4** 若 \(e^w = z\)，则称 \(w\) 为 \(z\) 的**对数**，记作 \(w = \operatorname{Ln} z\)。

由 \(z = re^{i\theta}\)，得：
\[
\boxed{\operatorname{Ln} z = \ln |z| + i \arg z + 2k\pi i = \ln r + i(\theta + 2k\pi)}, \quad k \in \mathbb{Z}
\]

**主值**（记作 \(\ln z\) 或 \(\operatorname{Ln}_0 z\)）：
\[
\boxed{\ln z = \ln |z| + i \arg z}, \quad -\pi < \arg z \leq \pi
\]

**关键性质**：
1. \(\operatorname{Ln} z\) 是**多值函数**，有无穷多个分支，彼此相差 \(2\pi i\) 的整数倍。
2. \(\ln z\) 是**主值分支**（即取 \(k=0\)，辐角在 \((-\pi, \pi]\) 内）。
3. 各单值分支在沿割线（通常取负实轴）的复平面上**解析**。
4. 导数：\((\operatorname{Ln} z)' = \frac{1}{z}\)（在各单值分支上）。

> 🔴 **注意**：\(\operatorname{Ln}(z_1 z_2) = \operatorname{Ln} z_1 + \operatorname{Ln} z_2\) 对**集合意义**成立，但对**主值**不一定成立。

### 五、幂函数 \(z^\alpha\)

**定义 2.4.5** 对任意复常数 \(\alpha\)，
\[
\boxed{z^\alpha = e^{\alpha \operatorname{Ln} z}}
\]

**分类讨论**：

| \(\alpha\) 的类型 | \(z^\alpha\) 的性质 |
|-----------------|-------------------|
| \(\alpha = n \in \mathbb{Z}^+\) | 单值解析函数，\(z^n\) 即为普通幂函数 |
| \(\alpha = -n\)（负整数） | 单值，除 \(z=0\) 外解析 |
| \(\alpha = \frac{p}{q}\)（有理数，既约） | \(q\) 值函数 |
| \(\alpha\) 为无理数或一般复数 | 无穷多值函数 |

**主值**（对应 \(\operatorname{Ln} z\) 取主值 \(\ln z\)）：
\[
\text{主值}(z^\alpha) = e^{\alpha \ln z}
\]

**导数**（在各单值分支上）：\((z^\alpha)' = \alpha z^{\alpha-1}\)。

**特殊值**（常考）：
- \(1^{\sqrt{2}} = e^{\sqrt{2} \operatorname{Ln} 1} = e^{\sqrt{2} \cdot 2k\pi i} = \cos(2\sqrt{2}k\pi) + i\sin(2\sqrt{2}k\pi)\)——无穷多值！
- \(i^i = e^{i \operatorname{Ln} i} = e^{i \cdot i(\frac{\pi}{2} + 2k\pi)} = e^{-(\frac{\pi}{2} + 2k\pi)}\)——无穷多个**实数**值！

### 六、反三角函数

**定义 2.4.6** 反三角函数可用对数函数表示：

\[
\boxed{\arcsin z = -i \operatorname{Ln}(iz + \sqrt{1-z^2})}
\]
\[
\boxed{\arccos z = -i \operatorname{Ln}(z + \sqrt{z^2-1})}
\]
\[
\boxed{\arctan z = \frac{i}{2} \operatorname{Ln}\frac{1-iz}{1+iz}}
\]

这些是多值函数（因含有 \(\operatorname{Ln}\)）。

**推导**：由 \(\sin w = \frac{e^{iw}-e^{-iw}}{2i} = z\)，解出 \(e^{iw}\)，再取对数即得。

## <a id="习题"></a>第二章习题精讲（对应题库 2.pdf）

### 第一部分：判断与填空（逐题解析）

**第1题**：「若 \(f(z)=u(x,y)+iv(x,y)\) 在区域 \(D\) 上满足柯西-黎曼条件，则它在区域 \(D\) 内解析。」

> ❌ **错误**。满足 C-R 方程只是解析的**必要条件**，还需 \(u, v\) 可微（偏导数连续）。反例：可构造满足 C-R 但偏导数不连续的函数，它并不可导。

**第2题**：「若 \(f(z)\) 在点 \(z_0\) 可导，则 \(f(z)\) 在点 \(z_0\) 解析。」

> ❌ **错误**。一点可导不意味着该点的邻域内可导。典型反例：\(f(z) = |z|^2 = z\bar{z}\)，在 \(z=0\) 处可导（导数 = 0），但在任何邻域内不解析（因为仅在原点满足 C-R 方程）。

**第3题**：「若 \(f(z)\) 在点 \(z_0\) 解析，则存在 \(\delta>0\)，使得 \(f(z)\) 在 \(U(z_0, \delta)\) 内解析。」

> ✅ **正确**。解析的定义本身就是存在一个邻域在该邻域内处处可导。注意这里说的是解析推出存在解析邻域——这是解析函数的基本性质。

**第4题**：「若 \(f(z)\) 在点 \(z_0\) 不连续，则 \(f(z)\) 在点 \(z_0\) 不可导。」

> ✅ **正确**。可导必连续，逆否命题成立：不连续必不可导。

**第5题**：「设 \(\ln z = 1 + \pi i\)，则 \(z = \_\_\_\)。」

> **解**：\(\ln z = \ln|z| + i\arg z = 1 + \pi i\)，故 \(\ln|z| = 1\)，\(\arg z = \pi\)。
> 因此 \(|z| = e^1 = e\)，\(\arg z = \pi\)，得 \(z = e \cdot e^{i\pi} = e \cdot (-1) = -e\)。
> 
> 答案：\(z = -e\)。

**第6题**：「当 \(z = \_\_\_\) 时，\(e^z\) 为实数。」

> **解**：\(e^z = e^x(\cos y + i\sin y)\) 为实数当且仅当虚部为零，即 \(\sin y = 0\)，\(y = k\pi\)，\(k \in \mathbb{Z}\)。
> 即 \(\operatorname{Im}(z) = k\pi\)。
> 
> 答案：\(\operatorname{Im}(z) = k\pi\)，\(k \in \mathbb{Z}\)。

**第7题**：「求 \(e^{-\frac{\pi i}{3}}\) 的值。」

> **解**：\(e^{-\frac{\pi i}{3}} = \cos\left(-\frac{\pi}{3}\right) + i\sin\left(-\frac{\pi}{3}\right) = \frac{1}{2} - i\frac{\sqrt{3}}{2}\)。

**第8题**：「求 \(\sin(1+i)\) 的值。」

> **解**：
> \[
> \sin(1+i) = \frac{e^{i(1+i)} - e^{-i(1+i)}}{2i} = \frac{e^{-1+i} - e^{1-i}}{2i}
> = \frac{e^{-1}(\cos 1 + i\sin 1) - e^{1}(\cos 1 - i\sin 1)}{2i}
> \]
> 整理得：
> \[
> \sin(1+i) = \sin 1 \cosh 1 + i \cos 1 \sinh 1
> \]
> 或直接使用公式 \(\sin(x+iy) = \sin x \cosh y + i \cos x \sinh y\)。

**第9题**：「\(\sin z\) 的周期为\_\_\_。」

> **解**：\(\sin(z+T) = \sin z\) 对所有 \(z\) 成立的最小正周期是 \(T = 2\pi\)。

**第10题**：「\((z+1)^5\) 的解析区域为\_\_\_，其导数为\_\_\_。」

> **解**：多项式函数在整个复平面上解析。解析区域：\(\mathbb{C}\)（整个复平面）。导数：\(5(z+1)^4\)。

**第11题**：「设 \(e^z = -1\)，则 \(z = \_\_\_\)。」

> **解**：\(-1 = e^{\pi i + 2k\pi i} = e^{(2k+1)\pi i}\)，故 \(z = (2k+1)\pi i\)，\(k \in \mathbb{Z}\)。

**第12题**：「\(e^z\) 的周期为\_\_\_。」

> **解**：\(e^{z+2\pi i} = e^z\)，周期为 \(T = 2\pi i\)。

**第13题**：「设 \(f(z)=u+iv\) 在区域 \(D\) 上，若 \(u\) 与 \(v\) 在 \(D\) 上有界，则 \(f(z)\) 在 \(D\) 上有界。」

> ✅ **正确**。\(|f| = \sqrt{u^2+v^2}\)，若 \(u, v\) 有界，则 \(|f|\) 有界。但需注意这里并没有说 \(f\) 解析——只是一个简单的模不等式问题。若 \(u, v\) 分别有界，则 \(|f| \leq \sqrt{M_u^2 + M_v^2}\)。

### 第二部分：证明与计算（逐题解析）

#### 第1题：连续性讨论

**题**：试问函数 \(\frac{1}{1+z^2}\) 在单位圆盘 \(|z|<1\) 内是否连续？是否一致连续？

> **解**：分母 \(1+z^2 = 0\) 当且仅当 \(z = \pm i\)，这两点在单位圆周上（\(|i| = |-i| = 1\)），不在开圆盘 \(|z|<1\) 内。因此在 \(|z|<1\) 内分母不为零，函数作为解析函数的商是解析的，从而连续。
> 
> 关于**一致连续性**：在开圆盘 \(|z|<1\) 内，函数连续但不一致连续。因为当 \(z\) 趋近于 \(\pm i\)（圆周边界点）时，函数趋于无穷，不存在连续模估计。

#### 第2题：不可微证明

**题**：证明函数 \(f(z) = |z|^2\) 除去在 \(z=0\) 外，处处不可微。

> **证明**：\(f(z) = |z|^2 = x^2 + y^2\)，所以 \(u = x^2+y^2\)，\(v = 0\)。
> 
> C-R 方程要求：\(u_x = v_y \Rightarrow 2x = 0 \Rightarrow x = 0\)；\(u_y = -v_x \Rightarrow 2y = 0 \Rightarrow y = 0\)。
> 
> 因此仅在原点 \(z=0\) 满足 C-R 方程。在原点处可直接用定义验证：
> \[
> f'(0) = \lim_{\Delta z \to 0} \frac{|\Delta z|^2}{\Delta z} = \lim_{\Delta z \to 0} \overline{\Delta z} = 0
> \]
> 所以在 \(z=0\) 处可导（导数为 0），但在任何 \(z \neq 0\) 处不满足 C-R 方程，从而不可导。
> 
> 由于原点可导但任何邻域内包含不满足 C-R 的点，故处处不解析。

#### 第3-5题：解析函数为常数的三种条件

**第3题**：若 \(f'(z) = 0\) 对 \(D\) 内每点成立，则 \(f\) 在 \(D\) 内为常数。

> **证明**：由 \(f'(z) = u_x + iv_x = v_y - iu_y = 0\)，得 \(u_x = v_x = u_y = v_y = 0\) 在 \(D\) 内恒成立，因此 \(u, v\) 均为常数，故 \(f\) 为常数。

**第4题**：若 \(|f(z)|\) 在 \(D\) 内为常数，则 \(f\) 在 \(D\) 内为常数。

> **证明**：设 \(|f(z)| = c\)（常数）。若 \(c=0\)，则 \(f(z) \equiv 0\)，显然为常数。
> 
> 若 \(c > 0\)，则 \(u^2 + v^2 = c^2\)。两边分别对 \(x, y\) 求偏导：
> \[
> 2u u_x + 2v v_x = 0, \quad 2u u_y + 2v v_y = 0
> \]
> 利用 C-R 方程 \(v_x = -u_y\)，\(v_y = u_x\) 代入：
> \[
> u u_x - v u_y = 0, \quad u u_y + v u_x = 0
> \]
> 这是关于 \(u_x, u_y\) 的齐次线性方程组。系数行列式 \(= u^2+v^2 = c^2 \neq 0\)，故 \(u_x = u_y = 0\)，从而 \(u\) 为常数。同理 \(v\) 为常数，故 \(f\) 为常数。

**第5题**：若 \(\operatorname{Re}[f(z)]\) 在 \(D\) 内为常数，则 \(f\) 在 \(D\) 内为常数。

> **证明**：设 \(u\) 为常数，则 \(u_x = u_y = 0\)。由 C-R 方程：\(v_y = u_x = 0\)，\(v_x = -u_y = 0\)，故 \(v\) 也为常数，因此 \(f\) 为常数。

#### 第7题：用C-R条件证明解析性/非解析性

**题**：试证 \(z^2, e^z, \sin z, \cos z\) 在复平面解析，而 \(\bar{z}, \bar{z}^2, e^{\bar{z}}, \sin \bar{z}, \cos \bar{z}\) 不解析。

> **证明思路**（以 \(z^2\) 和 \(\bar{z}\) 为例）：

**\(f(z) = z^2\)**：\(u = x^2-y^2\)，\(v = 2xy\)。
- \(u_x = 2x = v_y\) ✓
- \(u_y = -2y = -v_x\) ✓
- 偏导数连续，故在整个复平面解析。

**\(f(z) = \bar{z} = x - iy\)**：\(u = x\)，\(v = -y\)。
- \(u_x = 1\)，\(v_y = -1\)，不相等 ✗
- 在整个复平面不满足 C-R 方程，故处处不可导，不解析。

> **一般结论**：任何依赖于 \(\bar{z}\) 的函数在复平面上不解析（除非是常数）。

#### 第8题：极坐标C-R方程证明

**题**：证明在极坐标下 C-R 条件为 \(\frac{\partial u}{\partial r} = \frac{1}{r}\frac{\partial v}{\partial \theta}\)，\(\frac{\partial v}{\partial r} = -\frac{1}{r}\frac{\partial u}{\partial \theta}\)。

> **证明**：由链式法则：
> \[
> \frac{\partial u}{\partial r} = \frac{\partial u}{\partial x}\frac{\partial x}{\partial r} + \frac{\partial u}{\partial y}\frac{\partial y}{\partial r}
> = u_x \cos\theta + u_y \sin\theta
> \]
> \[
> \frac{\partial v}{\partial \theta} = v_x(-r\sin\theta) + v_y(r\cos\theta)
> = -r v_x \sin\theta + r v_y \cos\theta
> \]
> 
> 利用直角坐标 C-R 方程 \(u_x = v_y\)，\(u_y = -v_x\)：
> \[
> \frac{1}{r}\frac{\partial v}{\partial \theta} = -v_x\sin\theta + v_y\cos\theta = u_y\sin\theta + u_x\cos\theta = \frac{\partial u}{\partial r}
> \]
> 第二个公式类似可证。

#### 第9题：计算特殊值

**题**：求 \(e^{2+i}\)、\(i^i\)、\((-2)^2\) 的值。

> **(1) \(e^{2+i}\)**：
> \[
> e^{2+i} = e^2 \cdot e^i = e^2(\cos 1 + i\sin 1)
> \]
> 
> **(2) \(i^i\)**：
> \[
> i^i = e^{i\operatorname{Ln} i} = e^{i \cdot i(\frac{\pi}{2}+2k\pi)} = e^{-(\frac{\pi}{2}+2k\pi)}
> \]
> 主值：\(e^{-\pi/2}\)（取 \(k=0\)）。
> 
> **(3) \((-2)^2\)**：
> 这里指数 2 是整数，所以是单值的：
> \[
> (-2)^2 = 4
> \]

#### 第10题：对数计算

**题**：求 \(\operatorname{Ln}(1+i)\) 和 \(1^{\sqrt{2}}\) 的值。

> **(1) \(\operatorname{Ln}(1+i)\)**：
> \[
> 1+i = \sqrt{2}\left(\cos\frac{\pi}{4} + i\sin\frac{\pi}{4}\right)
> \]
> \[
> \operatorname{Ln}(1+i) = \ln\sqrt{2} + i\left(\frac{\pi}{4} + 2k\pi\right) = \frac{1}{2}\ln 2 + i\left(\frac{\pi}{4} + 2k\pi\right)
> \]
> 主值：\(\ln(1+i) = \frac{1}{2}\ln 2 + i\frac{\pi}{4}\)。
> 
> **(2) \(1^{\sqrt{2}}\)**：
> \[
> 1^{\sqrt{2}} = e^{\sqrt{2}\operatorname{Ln} 1} = e^{\sqrt{2} \cdot 2k\pi i} = \cos(2\sqrt{2}k\pi) + i\sin(2\sqrt{2}k\pi)
> \]
> 无穷多值！

#### 第11题：反三角函数的对数表达式

**题**：利用对数函数求出 \(\arcsin z\) 和 \(\arccos z\) 的解析表达式。

> **推导**：由 \(w = \arcsin z\) 即 \(\sin w = z\)。
> \[
> \frac{e^{iw} - e^{-iw}}{2i} = z
> \]
> 令 \(t = e^{iw}\)，则 \(t^2 - 2izt - 1 = 0\)。
> \[
> t = iz \pm \sqrt{1-z^2}
> \]
> \[
> e^{iw} = iz + \sqrt{1-z^2} \quad (\text{取主值分支})
> \]
> \[
> \arcsin z = -i \operatorname{Ln}(iz + \sqrt{1-z^2})
> \]
> 类似可得 \(\arccos z = -i \operatorname{Ln}(z + \sqrt{z^2-1})\)。

#### 第12题：复偏导数的 C-R 表述

**题**：设 \(u(x,y)\) 有偏导数，证明 \(\frac{\partial u}{\partial z} = \frac{1}{2}(u_x - iu_y)\)，\(\frac{\partial u}{\partial \bar{z}} = \frac{1}{2}(u_x + iu_y)\)，并证明 C-R 条件可写为 \(\frac{\partial f}{\partial \bar{z}} = 0\)。

> **证明**：由 \(x = \frac{z+\bar{z}}{2}\)，\(y = \frac{z-\bar{z}}{2i}\)，链式法则：
> \[
> \frac{\partial u}{\partial z} = u_x \frac{\partial x}{\partial z} + u_y \frac{\partial y}{\partial z} = u_x \cdot \frac{1}{2} + u_y \cdot \frac{1}{2i} = \frac{1}{2}(u_x - iu_y)
> \]
> \[
> \frac{\partial u}{\partial \bar{z}} = u_x \frac{\partial x}{\partial \bar{z}} + u_y \frac{\partial y}{\partial \bar{z}} = u_x \cdot \frac{1}{2} + u_y \cdot \left(-\frac{1}{2i}\right) = \frac{1}{2}(u_x + iu_y)
> \]
> 
> 对 \(f = u + iv\)：
> \[
> \frac{\partial f}{\partial \bar{z}} = \frac{1}{2}(u_x + iu_y) + \frac{i}{2}(v_x + iv_y) = \frac{1}{2}(u_x - v_y) + \frac{i}{2}(u_y + v_x)
> \]
> C-R 方程 \(u_x = v_y\)，\(u_y = -v_x\) 等价于 \(\frac{\partial f}{\partial \bar{z}} = 0\)。
> 
> 这是一个**极高观点**的结论：解析函数就是与 \(\bar{z}\) 无关的函数（在导数意义下）！

#### 第13-15题：多值函数的解析分支

**第13题**：在复平面上取上半虚轴作割线，取定 \(\sqrt{z}\) 和 \(\operatorname{Ln} z\) 在正实轴取正实值的解析分支，求它们在 \(z=i\) 处的值。

> **解**：割线是上半虚轴（即从原点出发沿正虚轴方向的射线）。
> 
> **\(\sqrt{z}\)**：要求 \(\sqrt{1} = 1\)（正实轴上取正值）。在割后的区域（即 \(\mathbb{C}\) 去掉上半虚轴），辐角范围为 \((-\frac{3\pi}{2}, \frac{\pi}{2}]\)。
> 
> \(i = e^{i\pi/2}\)，在此范围内 \(\arg(i) = \frac{\pi}{2}\)。
> 
> \[
> \sqrt{i} = e^{i\pi/4} = \frac{\sqrt{2}}{2} + i\frac{\sqrt{2}}{2}
> \]
> 
> **\(\operatorname{Ln} z\)**：\(\operatorname{Ln}(i) = \ln 1 + i\frac{\pi}{2} + 2k\pi i\)，主值该分支中 \(\arg(i)=\frac{\pi}{2}\)，故该分支在 \(z=i\) 的值为 \(\frac{\pi}{2}i\)。

#### 第16题：推理错误分析

**题**：找出推理错误：因为 \((-z)^2 = z^2\)，所以 \(2\operatorname{Ln}(-z) = 2\operatorname{Ln} z\)，因此 \(\operatorname{Ln}(-z) = \operatorname{Ln} z\)。

> **错误分析**：\(\operatorname{Ln}\) 是多值函数，\(\operatorname{Ln}(w^2) = 2\operatorname{Ln} w\) 对集合意义成立（即左右两边的值集合相等），但不能推出 \(\operatorname{Ln}(-z) = \operatorname{Ln} z\)。
> 
> 实际上，\(\operatorname{Ln}(-z) = \ln|z| + i\arg(-z) + 2k\pi i\)，\(\operatorname{Ln} z = \ln|z| + i\arg z + 2k\pi i\)，而 \(\arg(-z) = \arg z + \pi\)（当 \(\arg z\) 不在边界时），因此 \(\operatorname{Ln}(-z) = \operatorname{Ln} z + \pi i\)，不相等。
> 
> 推理中从 \(2\operatorname{Ln}(-z) = 2\operatorname{Ln} z\) 两边"除以 2"，忽略了对数函数多值性带来的集合相等与元素相等的区别。

#### 第19题：若 \(f(z)\) 和 \(\overline{f(z)}\) 均在 D 内解析，则 \(f(z)\) 为常数

> **证明**：设 \(f = u+iv\) 解析，满足 C-R：\(u_x = v_y\)，\(u_y = -v_x\)。
> 
> \(\overline{f} = u - iv\) 也解析，满足 C-R：\(u_x = (-v)_y = -v_y\)，\(u_y = -(-v)_x = v_x\)。
> 
> 联合：\(u_x = v_y\) 且 \(u_x = -v_y\)，故 \(u_x = v_y = 0\)；\(u_y = -v_x\) 且 \(u_y = v_x\)，故 \(u_y = v_x = 0\)。
> 
> 因此 \(u, v\) 为常数，\(f\) 为常数。

## 🔑 第二章核心定理与公式速查表

### 必背关键定理

| 定理 | 内容 |
|------|------|
| **C-R 方程（直角坐标）** | \(u_x = v_y\)，\(u_y = -v_x\) |
| **C-R 方程（极坐标）** | \(u_r = \frac{1}{r}v_\theta\)，\(v_r = -\frac{1}{r}u_\theta\) |
| **可导必要条件** | 可导 ⇒ C-R 成立 |
| **可导充分条件** | 偏导数连续 + C-R ⇒ 可导 |
| **解析充要条件** | 偏导数连续 + C-R ⇔ 在区域内解析 |
| **调和性** | 解析 ⇒ \(u, v\) 均为调和函数（满足 \(\Delta=0\)） |
| **解析 ⇒ 无穷阶可导** | 一经解析，拥有任意阶导数 |
| **C-R 的复形式** | \(\frac{\partial f}{\partial \bar{z}} = 0\) |

### 初等函数速查

| 函数 | 公式 | 周期 | 解析域 | 特殊值 |
|------|------|------|--------|--------|
| \(e^z\) | \(e^x(\cos y + i\sin y)\) | \(2\pi i\) | \(\mathbb{C}\) | \(e^{i\pi}=-1\) |
| \(\sin z\) | \(\frac{e^{iz}-e^{-iz}}{2i}\) | \(2\pi\) | \(\mathbb{C}\) | 零点：\(k\pi\) |
| \(\cos z\) | \(\frac{e^{iz}+e^{-iz}}{2}\) | \(2\pi\) | \(\mathbb{C}\) | 零点：\(\frac{\pi}{2}+k\pi\) |
| \(\operatorname{Ln} z\) | \(\ln\|z\| + i(\arg z+2k\pi)\) | — | 割线外 | \(\operatorname{Ln}(-1)=(2k+1)\pi i\) |
| \(z^\alpha\) | \(e^{\alpha\operatorname{Ln} z}\) | — | 视 \(\alpha\) 而定 | \(i^i = e^{-\pi/2+2k\pi}\) |

### 常见误区（考试选择题挖坑重点）

| 错误说法 | 正确理解 |
|----------|----------|
| \(|\sin z| \leq 1\) | ❌ 复数域无界，例如 \(\cos i \approx 1.543\) |
| \(|\cos z| \leq 1\) | ❌ 同上 |
| 可导 = 解析 | ❌ 一点可导不等于解析，需邻域条件 |
| 满足 C-R = 解析 | ❌ 还需偏导数连续（可微） |
| \(\operatorname{Ln}(z_1 z_2) = \operatorname{Ln} z_1 + \operatorname{Ln} z_2\) | ⚠️ 对主值不一定成立，仅在集合意义下成立 |
| \(z^\alpha\) 总是单值 | ❌ 视 \(\alpha\) 而定，一般多值 |
